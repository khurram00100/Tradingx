 Project Title
-----------------------------------------------------------------------------------------------------------------
TradingX
AI-Based Cryptocurrency, Forex & Gold Price Forecasting System using LSTM Neural Networks

 Overview
-------------------------------------------------------------------------------------------------------------------
TradingX is an advanced AI-powered trading assistant that:
Fetches real-time crypto, forex, and gold data
Trains deep LSTM neural networks
Predicts multi-hour future prices
Displays interactive charts
Includes AI chatbot + voice assistant
Stores data securely in SQLite database
Built completely in Python with PyTorch, this is a production-level AI desktop application.
